About

I am a member of technical staff at Anthropic. I am passionate about low-level systems, such as operating systems and virtualization, as well as their applications in sandboxing, performance, confidential computing, fuzzing, and reverse engineering. Previously, I was a Senior Staff Software Engineer at Databricks working on compute efficiency extensively across the serverless stack, where I optimized container startup latency by 7x using lazy-loading filesystem and memory snapshots, and built a virtualization stack for secure multi-tenant container execution across shared VMs. I had also spent 5 years at Microsoft Research as a Principal Researcher doing relevant systems security research, when I built a nested hypervisor for running legacy OSes in a confidential VM, found tens of VM-escape vulnerabilities in Hyper-V via hardware-assisted fuzzing, and created a reverse debugging system for production software crashes (and won the best paper award at OSDI!). Besides serious research, I developed an SGX-based system for protecting Windows PC games from piracy, without requiring source code changes or recompilation.
